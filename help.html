‚ùì Frequently Asked Questions

How do I create a new task?

+

To create a new task, go to the main app page and use the "Add New Task" form. Enter your task description, select a category and priority, optionally set a due date, then click "Add Task".

Can I edit tasks after creating them?

+

Yes! Click on any task to open the task details modal where you can edit the title, change priority, update category, or modify the due date.

How do I delete completed tasks?

+

Use the "Clear Done" button in the filters section to remove all completed tasks at once, or delete individual tasks using the delete button on each task.

Is my data saved automatically?

+

Yes, all your tasks and settings are automatically saved to your browser's local storage. Your data persists between sessions and is completely private.

Can I export my tasks?

+

Absolutely! Go to Settings > Data Management and use the "Export Tasks" button to download your tasks as a JSON file for backup or transfer.

What do the priority colors mean?

+

üî¥ Red = High Priority (urgent tasks), üü° Yellow = Medium Priority (important tasks), üü¢ Green = Low Priority (can be done later).

How do I use the search feature?

+

Use the search box in the filters section to find tasks by typing keywords from the task title. The search is case-insensitive and updates results in real-time.

Can I use TaskMaster Pro offline?

+

Yes! TaskMaster Pro works completely offline. All data is stored locally in your browser, so you can manage your tasks without an internet connection.
